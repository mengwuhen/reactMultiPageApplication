 ### 2018.2.1
  对项目的目录结构做出调整，并引入react版本的 echart 和 g2 组件库

  ## 工程目录说明

> --app 主目录
    >> component    react组件目录
      >>> pages    页面目录 
    >>--public    公共目录 主要放 js css img
      >>> --js    公共js文件
      >>> --css  css目录
      >>> --img   图片目录
> --devBuild    webpack 开发环境下 打包输出目录
> --config    打包配置目录
       >> --webpack    webpack配置目录
> --entryBuild    webpack 打包入口文件目录

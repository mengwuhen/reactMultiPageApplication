## 打个 tag
  生产环境代码需要有几个特点:
    ……文件体积尽量的小
    ……浏览器缓存!如果修改,如何让浏览器重新拉取
    ……请求数尽量少

  ……我们把文件打包到 根目录下 pc文件夹下 

  ……文件都自动加上了 hash值,便于浏览器缓存,修改后这个hash会变,浏览器会重新获取. 
  ……为什么没有 图片文件夹?
  ……这是因为你的那张百度logo图片大小没有超过8192,被base64转码了,减少了一次请求.

  运行 npm run prod 得到生产环境的代码 （^v^ 离成功更进一步了）
